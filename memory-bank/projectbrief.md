# Проект: Simple Python Website

## Цель проекта
Прототипирование современных веб-сервисов с использованием простой, но функциональной архитектуры на Python. Проект предназначен для образовательных целей и быстрого прототипирования, а не для production-развертывания.

## Ключевые требования
- Веб-сервер на основе Bottle
- Аутентификация через MD5 хэширование и подписанные cookie
- База данных SQLite
- SPA фронтенд на Vue.js 3 с Vue Router 4 и Bootstrap 5
- REST API для взаимодействия фронтенда и бэкенда
- Поддержка различий ролей пользователей (admin/user)
- Возможность администрирования пользователей и данных

## Нефункциональные требования
- Простота и понятность кода для образовательных целей
- Минимальные зависимости
- Возможность публикации через ngrok
- Конфигурация через переменные окружения

## Границы проекта
Проект НЕ предназначен для:
- Production-развертывания
- Хранения чувствительных данных
- Высоконагруженных сценариев
- Масштабирования на несколько инстансов
