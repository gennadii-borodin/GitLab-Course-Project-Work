# Контекст продукта

## Проблема
Необходим простой и понятный каркас для быстрого прототипирования веб-сервисов с аутентификацией и базовым администрированием, который можно использовать для образовательных целей и демонстраций.

## Решение
Simple Python Website предоставляет:
- Готовую архитектуру с разделением на бэкенд (Bottle + SQLite) и фронтенд (Vue.js SPA)
- Систему аутентификации с поддержкой регистрации, логина и ролевой модели
- REST API для взаимодействия с данными
- Административные интерфейсы для управления пользователями и контентом

## Целевая аудитория
- Студенты и начинающие разработчики, изучающие веб-разработку
- Преподаватели, готовящие образовательные курсы
- Разработчики, которым нужно быстро создать прототип веб-сервиса

## Пользовательский опыт
- Чистый, современный интерфейс на основе Bootstrap 5
- SPA-навигация без перезагрузки страницы
- Интуитивные формы для регистрации и входа
- Разделение интерфейса для обычных пользователей и администраторов
- Информативные сообщения об ошибках и успехе операций

## Критерии успеха
- Простота развертывания (минимальное количество настроек)
- Понятность кода для начинающих разработчиков
- Функциональность, достаточная для создания рабочих прототипов
- Легкость расширения и кастомизации
